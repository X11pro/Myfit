# Vision de producto

## Propuesta

`Myfit` debe responder una pregunta diaria simple: `Estoy comiendo lo adecuado para mi objetivo teniendo en cuenta entrenamiento, trabajo y vida diaria?`

## Diferenciacion

- Balance energetico realista, no solo conteo de calorias.
- Foco en proteina, calorias, azucar y adherencia.
- Integracion con `Health Connect` y `HealthKit`.
- Modo de trabajo fisico para usuarios fuera del perfil tipico de oficina.
- IA para reducir friccion, no para aparentar precision medica.

## Usuario inicial

- Persona que entrena fuerza o fitness general.
- Puede tener trabajo sedentario o fisico.
- Quiere registrar comida rapido.
- Necesita recomendaciones entendibles y accionables.

## Principios de producto

1. Confirmacion humana antes de guardar estimaciones dudosas.
2. Privacidad fuerte y permisos minimos.
3. Explicaciones simples sobre calculos y recomendaciones.
4. MVP pequeno pero utilizable todos los dias.
