# User stories iniciales

## Onboarding

- Como usuario, quiero indicar mi objetivo corporal para que la app ajuste calorias y proteina.
- Como usuario, quiero conectar `Health Connect` o `HealthKit` para no cargar entrenamientos a mano.

## Nutricion

- Como usuario, quiero registrar una comida en texto libre para guardarla rapido.
- Como usuario, quiero escanear un barcode para cargar macros sin escribir.
- Como usuario, quiero sacar una foto y corregir la porcion antes de guardar.

## Balance diario

- Como usuario, quiero ver si voy en deficit, mantenimiento o superavit para ajustar el resto del dia.
- Como usuario, quiero saber cuanto me falta de proteina para cumplir mi objetivo.

## Entrenamiento

- Como usuario, quiero importar sesiones desde la plataforma de salud del sistema.
- Como usuario, quiero registrar sets manuales del gym cuando no haya datos importados.

## Coach

- Como usuario, quiero una recomendacion corta y segura basada en mis datos del dia.
- Como usuario, quiero que la app priorice comida normal antes que suplementos.
