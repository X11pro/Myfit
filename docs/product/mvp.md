# MVP

## Objetivo

Entregar una primera version que permita registrar comida, importar actividad basica, calcular balance diario y mostrar una recomendacion simple.

## Alcance incluido

- Registro y onboarding de usuario.
- Perfil corporal y objetivo.
- Registro manual de alimentos.
- Escaneo de barcode con `Open Food Facts` y fallback `USDA`.
- Foto de comida con confirmacion manual.
- Dashboard diario con calorias, proteina, azucar y balance estimado.
- Importacion desde `Health Connect` y `HealthKit`.
- Registro manual de entrenamiento de gimnasio.
- Coach diario con recomendaciones cortas y seguras.

## Alcance excluido por ahora

- Tracking medico.
- Precision clinica de fotos.
- Dependencia directa de `Zepp`.
- Uso central de datos `Strava` para IA.
- GPS/rutas avanzadas.
- Recomendaciones agresivas de suplementacion.

## KPI iniciales

- Tiempo de registro manual de comida menor a 20 segundos.
- Flujo de foto con confirmacion en menos de 45 segundos.
- Usuario puede entender el estado diario sin abrir mas de una pantalla.
- Beta cerrada con 10 a 20 usuarios activos.
